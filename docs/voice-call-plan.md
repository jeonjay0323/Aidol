# AI 연습생 음성통화 기획안

## 목표

연습생 캐릭터와 실시간 음성 대화가 가능한 인터랙션 시스템 구현

---

## 기술 스택

| 역할 | 선택 | 이유 |
|------|------|------|
| 실시간 대화 AI | Gemini 3.1 Flash Live (Google AI Studio) | 무료 티어 제공 |
| STT / TTS | Gemini Live 내장 | 별도 서비스 불필요 |
| 아바타 | 1단계 없음 → 추후 D-ID / Simli 추가 | 모듈식 설계 |
| 마이크 / 스피커 | 브라우저 WebRTC | 무료 |

---

## 아키텍처

```
사용자 마이크
    ↓
Gemini 3.1 Flash Live
    ↓ (연습생 성격 프롬프트)
음성 응답 스트리밍
    ↓
[추후] 아바타 립싱크 추가
```

### 모듈식 설계 원칙

아바타를 나중에 붙일 수 있도록 TTS 출력과 아바타 렌더링을 분리한다.

```python
audio_output = tts.synthesize(text)
avatar.speak(audio_output)  # 추후 이 레이어만 추가
```

## 개발 로드맵

### 1단계 — 음성 대화 프로토타입
- Google AI Studio API 키 발급
- Gemini 3.1 Flash Live WebSocket 연결
- 브라우저 마이크 입력 → 실시간 음성 응답

### 2단계 — 연습생 캐릭터 적용
- 연습생별 시스템 프롬프트 작성 (성격, 말투, 배경 스토리)
- 연습생 선택 UI

### 3단계 — 아바타 연결 (추후)
- D-ID 또는 Simli API 연동
- 연습생 프로필 사진 기반 실시간 립싱크
